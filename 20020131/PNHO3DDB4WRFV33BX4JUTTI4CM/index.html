綁架富豪警匪駁火18響
《蘋果》貼身影槍匪

【本報訊】尖沙嘴鬧市昨晨發生綁架警匪槍戰案件。八十歲老商人兼馬主蘇澤棠，昨晨由司機駕車離開又一村寓所時，遭兩名持槍匪徒強行登車挾持，並迫令事主致電公司勒索二十萬元贖金，警方接獲報告後展開圍捕，先在尖沙嘴街頭與賊人兩度駁火共十八響，一名警署警長中彈受傷。商人及司機獲救無恙。至晚上，警方在上環港澳碼頭搜查時，分在碼頭及船上，拘捕三名疑涉案男子，部分持雙程證。

事件中，《蘋果》記者第一時間到尖沙嘴現場採訪，一度與賊人相距僅十呎，警匪駁火時子彈橫飛，驚險萬分。賊人輾轉逃去旺角，逾百警員進行圍剿，最後無功而還。

警昨晚拘捕三疑犯
直至昨晚，警方獲得事主提供的資料，相信除兩名綁匪外，並有至少一名幕後首腦，並取得三名疑人的姓名及資料交給入境處，防止疑人離境，至晚上九時，上環信德碼頭入境處果然發現一名準備往澳門的持雙程證男子為被通緝者，立即通知警方，警方迅速將該名姓黎的二十六歲男子拘捕。同時，警方懷疑另兩名可疑人已在往澳門的噴射船上，立即派出數十名警員荷槍實彈搜船，結果再拘兩人。
對於今次事件，政務司司長曾蔭權強調，香港是一個很安全的城市，而且香港警方對緝拿這些悍匪很有經驗，政府一定不能夠容忍這些暴行在香港發生。

僅索20萬疑有內情
消息稱，兩名綁匪操內地口音，不排除是受僱來港的旗兵，同時，賊人挾持事主在車上期間，曾用無線電話與一名懷疑是幕後主腦的人物通話，而且勒索金額只有二十萬元，因此，警方相信此案並非單純的綁架勒索案，並懷疑有人隱瞞真相。
被綁架的八十歲事主蘇澤棠，與家人同住又一村地錦路七號一幢花園洋房，蘇本身是多匹馬的馬主，並在中環砵甸乍街開設一間重型五金出入口公司，六十年代已是億萬富豪。他每日習慣由司機駕駛其棗紅色車牌CT50的凌志房車接送上班。
昨晨十時許，事主如常步出寓所大閘，準備登上私家車時，兩名匪徒突然衝前，並向天開槍一響示警，然後將事主推上其座駕的後座，兩匪登車分坐前後座，分別持槍指嚇事主及其司機，迫令開車。
至十一時許，事主公司的職員接獲蘇澤棠親自來電，透露自己被綁票，並要求職員籌措二十萬元，及在指定時間送到尖沙嘴柏麗大道交款，之後職員通知事主兄長蘇澄洲及報警，警方接報後即通報全港各區巡警留意事主的凌志房車。至中午十二時半，有巡警在彌敦道近北京道發現事主房車，立即通報上峯，而此時海防道、柯士甸道每個街口都有警員布防。
當時，兩名賊人一直在車上持槍威脅事主及司機，當車輛駛至近柯士甸道交通燈位時，事主車輛塞在中間，此時，《蘋果》記者駕駛電單車剛駛至，並恰巧停在事主汽車後面。

匪與追兵兩度槍戰
據記者目擊，當時，四名警員接報而至，分別在前後包抄，兩名警長在後面以一輛巴士作掩護，擎槍一步一步接近事主車輛，而另兩警則在前面進迫，此時，車上賊人已發覺有警員出現，立即棄下兩名事主下車，並向警員連開四槍，而警員還火六槍，均沒有傷人，只有一輛巴士擋風玻璃碎裂。事後警方在車上救出兩名事主，他們均沒有受傷。
兩賊人事後逃入山林道，再度與追至的警員駁火，賊人一槍擊中一名警署警長的左手，警員再還火七槍，但終被賊人逃脫。警方事後在橫巷撿獲賊人遺下的一支九MM蘇製曲尺手槍，警方發現尚有七彈在槍內，相信賊人開兩槍後槍膛卡彈，被迫棄槍而逃。
目擊者稱，一賊在覺士道登上的士向旺角逃去，另一則經過一間小學向德成街方向逃去，警方事後在現場撿獲一件皮褸及一頂鴨舌帽。警方一直緊追賊人的士至亞皆老街，賊人在近花園街下車逃入砵蘭街與快富街交界一條後巷。警方派出逾百警員荷槍實彈封鎖搜捕，至下午五時，相信賊人已逃脫收隊。
