AV陳列室：
天要落雨舊款DV機要落價！

就好似落雨呢個自然現象，當新機不再新就必然要冧價！呢個係永遠改變唔到嘅現實，就算近來日圓高企，都一樣有電器舖趁暑假將DV機冧價促銷，不過令人感到最詫異嘅，都係部SONYDCR-IP55E，新機落地個零月，已經馬上俾人急降身價千多元！想掃平DV機？𠵱家正是時候！

記者：朱崇德　攝影：陳盛臣
ImageGirl：CoCo

SONY最新Dream機勁冧$1,400
離譜！IP55相信是現在不少SONY迷的Dream機，但估唔到新機落地個多月，機價已跌到一萬頭的價位！
IP55一直以佢手柄式嘅復古外形設計，深得用家歡心，佢係SONY第二部使用MICROMV細帶格式嘅DV機，機身極度小巧，加上107萬像素的感光元件和10倍光學變焦的蔡司鏡頭（另加120倍數碼變焦），將生活品味和最新攝影科技合而為一！
2.5吋的觸屏式顯示器，令機身功能操作變成Palm一樣，輕輕「篤下」就搞掂，還備有藍芽組件，配合藍芽手機，就可以將剛拍下來的MPEG動態片段，經互聯網電郵給遠方親朋。此機存貨量唔會多過十部，如果是心頭好就一定快手！

Profile
CCD：1/4.7吋（107萬像素）
變焦：10倍光學、120倍數碼
尺寸：60（寬）×73（高）×128.5（深）mm
重量：435克

SONY迷必買實用型DV機
DCR-PC101E是今年4月新出的機款！是SONY使用傳統MiniDV帶的機種中最小巧的一部，重量只有480克，比DCR-PC115/120足足輕了100克！感光元件用1/4.7吋的CCD，動態及靜態影像的解像度，分別為69萬及100萬像素，算是不過不失，SONYDV機迷必買。

Profile
CCD：1/4.7吋（107萬像素）
變焦：10倍光學、120倍數碼
尺寸：55（寬）×105（高）×99（深）mm
重量：480克

Panasonic37mm大炮王
PanasonicNV-GX7只可以用大件夾抵食嚟形容！使用了一枚133萬像素的CCD，動態拍攝時的像素仍然能達69萬像素的高水平，不過最令拍友歡迎的，還是那支37mm口徑的10倍光學變焦LEICA鏡頭，影像質素細膩兼有保證！去年底剛推出時，日本本土曾一度賣到斷晒貨，𠵱家機王要下放身段，減價應市，真係入得過！拍攝靜態影像亦是NV-GX7的強項，123萬有效像素再配以一個沿着鏡頭弧形移動的閃光燈，真係不得不讚讚設計者的花巧心思。

Profile
CCD：1/4吋（133萬像素）
變焦：10倍光學、500倍數碼
尺寸：71（寬）×93（高）×139（深）mm
重量：550克

特靚畫質JVC前代機王
GR-DVP7是JVC去年直身民用DV機中的王牌之作，小巧機身加鎂合金材質的堅固外殼，曾一度成為全球最細的DV機！儘管全球最細紀錄已被新出嘅DV機追過，不過亦無減它機王的強勢本色；機身配置有一枚1/3.8吋102萬像素的CCD，拍攝動態影像時，有效像素最高可達89萬，大大高於同級DV機的實際有效像素62-71萬。GR-DVP7還特別加了一項「畫素倍密度轉換」功能，靜態影像的最高解像度可達1,280×960（約122萬像素），當埋佢係數碼相機都得！

Profile
CCD：1/3.8吋（102萬像素）
變焦：10倍光學、200倍數碼
尺寸：（寬）×115（高）×80（深）mm
重量：350克

PROGRESSIVESCAN超穩定畫質
CANONDM-MV4i機身體積細小，去年7月推出時贏得全球最細DV機的榮譽。今次5部冧價機中，佢係唯一行ProgressiveScan的DV機款，從動態影像輸出上看，1/4吋CCD提供68萬像素的影像好似平平無奇，不過動態影像每秒的格數可達25格，令畫面播放及定格表現方面更趨穩定！此外，鏡頭還備有RGB濾鏡功能，並能將光源分拆為最基本的三原色，然後再在DV帶中作還原儲存，影像質素當然超細膩啦！

Profile
CCD：1/4吋（80萬像素）
變焦：10倍光學、40倍數碼
尺寸：48（寬）×106（高）×86（深）mm
重量：390克
